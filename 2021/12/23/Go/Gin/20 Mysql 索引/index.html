Mysql 索引

发表于2021-12-23|更新于2021-12-28|GoGin框架
|阅读量:

MySQL 索引的建立对于 MySQL 的高效运行是很重要的，索引可以大大提高 MySQL 的检索速度。

如果没有索引，执行查询时候必须从第一条记录开始，扫描整个表的记录，直到符合要求的记录。如

果有了索引，mysql 无需扫描任何记录即可顺序找到目标记录的位置。简单说来， 索引就是提高查

找数据速度，数据量越多，效果越明显。

Mysql 中常见的索引类型有普通索引、唯一索引、全文索引、空间索引 Spatial
