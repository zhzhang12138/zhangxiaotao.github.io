一、区分py文件的两种途径

为了区别同一个文件的不同用途，每个py文件都内置了__name__变量，该变量在py文件被当做脚本执行时赋值为”main“，在py文件被当做模块导入时赋值为模块名。

作为模块spam.py的开发者，可以在文件末尾基于__name__在不同应用场景下值的不同来控制文件执行不同的逻辑。
