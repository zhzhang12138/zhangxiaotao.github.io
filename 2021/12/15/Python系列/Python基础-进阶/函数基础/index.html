名称空间与作用域

名称空间与作用域的关系是在函数定义阶段（扫描语法时）就确立的，与什么时候调用以及调用位置无关
namespace名称空间：存放名字的地方
作用域

在函数内，无论嵌套多少层，都可以查看到全局作用域的名字，若要在函数内修改全局名称空间中名字的值，当值为不可变类型时，则需要用到global关键字

对于嵌套多层的函数，使用nonlocal关键字可以将名字声明为来自外部嵌套函数定义的作用域（非全局）
